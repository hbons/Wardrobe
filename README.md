## 🧩 Features

- 🎨 Install **GNOME Shell** themes
- 🧵 Manage **GTK3 / GTK4** themes
- 🖼️ Browse and apply **wallpapers**
- 🖱️ Install **icon** and **cursor** themes
- 🌐 Themes are sourced from [OpenDesktop](https://www.opendesktop.org/)

## ⬇️ Installation

### Flathub (recommended)

```bash
flatpak install flathub io.github.swordpuffin.wardrobe
```

## 🤝 Contributing

All contributions are welcome as Wardrobe is still in active development.

Run the following commands in a terminal:

(Make a folder named `Projects` in your home directory if it doesn't already exist)

```bash
cd Projects
git clone https://github.com/SwordPuffin/Wardrobe
```

Then, in [Builder](https://apps.gnome.org/Builder/) you can add it to your projects.
